# send_reminders
sends reminders to people when they don't do their fish care chores. Reads data from [this](https://docs.google.com/spreadsheets/d/1pVwqyetFLGVl_2qQ40qCH0Nvhe7ODzKC7J_oyQsiOQg/edit#gid=0) Google spreadsheet. Written based on code [here](https://www.twilio.com/blog/2017/02/an-easy-way-to-read-and-write-to-a-google-spreadsheet-in-python.html) and [here](https://automatetheboringstuff.com/chapter16/).
